## Инструкция по запуску проекта
### 1. Запуск проекта
Для запуска проекта выполните в терминале следующую команду:
```bash
go run app/main.go
```

### 2. Вывод пояснений через флаг
Можно добавить флаги:
* `go run app/main.go --info`&emsp; &mdash; &emsp;флаг для вывода информации о проекте
* `go run app/main.go --help`&emsp; &mdash; &emsp;флаг для вывода инструкции по использованию
